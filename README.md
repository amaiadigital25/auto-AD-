1. Crear una carpeta llamada amaia-bot
2. Guardar todos los archivos
3. Abrir CMD dentro de la carpeta
4. Ejecutar:

npm install

5. Después ejecutar:

npm start

6. Va a aparecer un código QR
7. Abrí WhatsApp en tu celular
8. Entrá a Dispositivos vinculados
9. Escaneá el QR
10. El bot queda funcionando
